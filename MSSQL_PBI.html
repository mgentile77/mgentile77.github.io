Example 2: Historical Review

Next is an example of a report with data duplicated from the previous report. This report is a historical account of production totals and does not require data duplication. Additionally, when the query was run for this report, it took over 20 minutes to complete on a production-critical database and was run every hour. I optimized the query, reducing run time to less than two minutes, and removed the scheduling because stakeholders determined that the report did not need a scheduled refresh.
